Сайт справочник движений классического танца.
фронт:
всё в светло-бежевых оттенках
на главном экране шапка с названием, справа кнопка входа и регистрации, если навести на эту кнопу, появляется надпись "вход/регистрация", если пользователь не авторизован, и "Профиль", если авторизован. При нажатии переход на страницу пользователя.
слева от шапки значок стрелочки, нажав на который выплывает шторка, в которой есть пункты: "Главная", "поиск", "алфавитный указатель", "настройки сайта". каждый пункт ведет на свою страницу. правее кнопки со стрелочкой окно "поиск" и кнопочка рядом "найти". Шапка по горизонтали размером с весь экран, а по вертикали занимает 1/10 часть экрана. Шапка сохраняется на всех страницах, кроме регистрации и входа.

страница пользователя:
Если человек не авторизован, открывается страница с регистрацией или входом. Там два поля для ввода логина и пароля и кнопка для регистрации. При регистрации поля для ввода с надписями над окошками: "логин", "пароль", "почта", "имя пользователя". После регистрации данные переходят в базу данных. Для входа так же используется та же база данных. После авторизации на странице пользователя появляется слева шторка ("Избранное", "настройки профиля", "настройки сайта", "связаться с нами", "выход") (при нажатии на всё, кроме выхода, идет переход на страницу выбранного. При нажатии на "выход" выплывает окно "Вы точно хотите выйти из аккаунта" и два варианта ответа "Да" и "Нет". Если пользователь нажмет Да, пользователь выходит из акаунта, если нет, окно закрывается, пользователь остаётся в своем аккаунте). На главной части страницы приветствие по имени типа "Здравствуйте, {имя}"

"Главная" страница (при входе на сайт)
под шапкой идет информация о сайте, о классическом танце. Страница может опускаться ниже, но шапка остаётся стабильной. В самом низу страницы можно найти информацию о разработчике и контактные данные для связи. Там же окошечко для отправки сообщения об ошибки. Внутри окошечка написано: "Если заметили ошибку или хотите что-то добавить, напишите об этом здесь". После нажатия на кнопки снизу этого окна, сообщение с помощью тг-бота отправляется разработчику. 

страница "поиск":
посередине под шапкой поле окно для ввода текста "поиск" длиной на весь экран. оставляя место для кнопки "найти". После ввода искомого в поле для поиска, снизу появляются движения, которые подходят по поиску. Если совпадений нет, предлагает, либо похожие по написанию движения, либо показывает историю поиска. Также есть ссылка на страницу для "поиска по видео", где кнопка "загрузить видео" и "Режим реального времени". Ниже информация о таком виде поиска (пока пусто)

страница "алфавитный указатель": 
Под шапкой расположены фильтры для поиска в ряд. 1 тип фильтров: На русском языке, на французском языке. 2-й тип: По алфавиту по возрастанию, по алфавиту по убыванию, недавние (которые ранее открывались пользователем). 3-й тип: Позиции ног, прыжки, базовые движения. 
После фильтров идет список из движений: (сначала идёт названия на русском, в скобках на французском (для 1-го типа фильтров), а перед перечислением, классификации движений (для 3-го типа фильтров))

страница "настройки сайта":
Пока просто пустая страница

страница конкретного движения:
(При нажатии на название движения со страницы поиска, алфавитного указателя, истории поиска, избранного открывается страница про это движение) Под шапкой идет по центру заголовок - название движения. Ниже информация о движении (пока для каждого написать "информация о {название движения}"), поле для фото этого движения. Кнопочка "оценить выполнение", которая пока никуда не ведет. Поле для ввода текста "Добавьте заметку о движении", и рядом кнопка "сохранить заметку", при нажатии на которую на странице с движением появляется окошечко "Заметки:", и дальше введенный текст пользователя. Этот текст пользователь может в последствии изменить. Если пользователь не авторизован, появляется надпить "Чтобы добавить заметку о движении, авторизуйтесь". И кнопочка в виде сердечка, при наведении мышки на которую появляется надпись "Добавить в избранное", при нажатии на которую движение добавляется в список "избранное" для авторизованного пользователя. Если пользователь не авторизован, появляется надпить "Чтобы добавить движение в избранное, авторизуйтесь". Если движение добавлено в избранное, сердечко становится цветным, и если на него навести мышкой, будет надпись "Удалить из избранного"


страница "Избранное":
Движения появляются на этой странице, после того, как авторизованный пользователь нажал на странице движения на кнопку "Избранное". При нажатии на название движения, пользователь переходит на страницу движения.

страница "настройки профиля":
Под шапкой заголовок "Настройки профиля". Дальше идет окно с Логином пользователя. Рядом с ним текст-кнопка "Сменить логин". Аналогично с почтой и именем пользователя. В самом низу надпись "Выйти из профиля", после нажатия на которую выплывает окно "Вы точно хотите выйти из аккаунта" и два варианта ответа "Да" и "Нет". Если пользователь нажмет Да, пользователь выходит из аккаунта, если нет, окно закрывается, пользователь остаётся в своем аккаунте)

Страница "Связаться с нами":
Просто информация о разработчиках, способы связаться (Почта, тг-бот)
